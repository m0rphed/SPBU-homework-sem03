**УСЛОВИЕ ЗАДАЧИ**:

- Есть N тарелок спагетти, N вилок и N философов.
- Философ может думать и есть.
- Чтобы есть, философу нужны две вилки.
- Пример — транзакция, переводящая деньги со счёта на счёт.
